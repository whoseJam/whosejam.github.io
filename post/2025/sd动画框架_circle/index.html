Circle

继承自 sd.BaseSVG，封装了 circle 这种 SVG 元素。

API 列表 

class Circle extends BaseSVG {
    constructor(target: SDNode | RenderNode);

    r(): number;
    r(r: number): this;
}

r 

class Circle {
    r(): number;
    r(r: number): this;
}

用于管理圆的半径。对于圆而言，其长宽总是相等的，当设置其中之一时，另外那个也会随之改变，无论改变长还是宽，本质都是在改变半径。圆的默认半径为 20。
