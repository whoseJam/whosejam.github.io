1.MoveSelector

MoveSelectorдё»иҰҒзҡ„еҠҹиғҪжҳҜжҢҮеҜјд»ҺдёҖдёӘзҠ¶жҖҒпјҢеҰӮдҪ•иҺ·еҸ–еҲ°е…¶йӮ»еҹҹзҠ¶жҖҒпјҢд»ҘдҫҝдјҳеҢ–з®—жі•йҒҚеҺҶиҝҷдәӣйӮ»еҹҹзҠ¶жҖҒпјҢдёҖж¬ЎиҺ·еҸ–йӮ»еҹҹзҠ¶жҖҒзҡ„иҝҮзЁӢиў«з§°дёәдёҖж¬ЎMove

дёәдәҶеҲӣе»әдёҖдёӘMoveпјҢдёҖдёӘMoveSelectorйңҖиҰҒйҖүжӢ©дёҖдёӘжҲ–иҖ…еӨҡдёӘPlanning EntityжҲ–иҖ…Planning ValueжқҘMoveпјҢжҲ‘д»¬жҠҠиҫ…еҠ©жҲ‘д»¬йҖүжӢ©Planning Entityе’ҢPlanning Valueзҡ„дёңиҘҝеҲҶеҲ«еҸ«еҒҡEntitySelectorе’ҢValueSelector

дёҖдёӘMoveSelectorйҖҡеёёз”ұEntitySelectorе’ҢValueSelectorпјҢд»ҘеҸҠе…¶д»–зҡ„MoveSelectorз»„еҗҲиө·жқҘпјҢдҫӢеҰӮпјҡ

<unionMoveSelector>
    <changeMoveSelector>
        <entitySelector>
            ...
        </entitySelector>
        <valueSelector>
            ...
        </valueSelector>
    </changeMoveSelector>
    <swapMoveSelector>
        ...
    </swapMoveSelector>
</unionMoveSelector>

д»–д»¬жһ„жҲҗдәҶдёҖдёӘSelector TreeпјҢиҝҷдёӘж ‘зҡ„ж №дјҡиў«жіЁе…ҘеҲ°дјҳеҢ–з®—жі•зҡ„е®һзҺ°дёӯпјҢеңЁжҜҸдёҖж¬Ўиҝӯд»Јдёӯиў«и°ғз”Ё

2.еҹәзЎҖMoveSelector

changeMoveSelector

ChangeMoveйҖүжӢ©дёҖдёӘPlanning Entityзҡ„дёҖдёӘPlanning VariableпјҢж”№еҸҳиҜҘPlanning VariableдёәдёҖдёӘж–°еҖј

e <- Select a Planning Entity
v <- Select a Planning Variable of e
e.v <- new value in range(v)

swapMoveSelector

SwapMoveйҖүжӢ©дёӨдёӘдёҚеҗҢзҡ„Planning EntityпјҢдәӨжҚўе®ғд»¬жүҖжңүзҡ„Planning Variable

e1 <- Select a Planning Entity
e2 <- Select a Planning Entity
swap(e1, e2)

жңүж—¶еҸҜиғҪе№¶дёҚеёҢжңӣдәӨжҚўдёӨдёӘPlanning Entityзҡ„жүҖжңүPlanning VariableпјҢеҸҜд»ҘиҝӣиЎҢеҰӮдёӢй…ҚзҪ®пјҡ

<swapMoveSelector>
    ...
    <entitySelector>
        <entityClass>...Lecture</entityClass>
        ...
    </entitySelector>
    <secondaryEntitySelector>
        <entityClass>...Lecture</entityClass>
        ...
        <nearBySelection>...</nearBySelection>
    </secondaryEntitySelector>
    <variableNameINcludes>
        <variableNameInclude>room</variableNameInclude>
        <variableNameInclude>...</variableNameInclude>
    </variableNameINcludes>
</swapMoveSelector>

secondaryEntitySelectorе№¶дёҚдёҖе®ҡйңҖиҰҒпјҢеҰӮжһңе®ғжІЎжңүиў«й…ҚзҪ®пјҢйӮЈд№ҲдёӨдёӘиў«йҖүдёӯзҡ„Planning EntityйғҪз”ұentitySelectorжҢҮе®ҡ

еҰӮжһңдёҖдёӘжҲ–еӨҡдёӘvariableNameIncludeиў«й…ҚзҪ®дәҶпјҢйӮЈд№Ҳе°ұдёҚжҳҜжүҖжңүзҡ„Planning VariableйғҪдјҡиў«дәӨжҚўпјҢиҖҢд»…д»…жҳҜиў«variableNameIncludeйҖүдёӯзҡ„еұһжҖ§дјҡиў«дәӨжҚў

6.SelectorFeatures

cacheType

дёҖдёӘSelectorзҡ„cacheTypeеҶіе®ҡдәҶдёҖдёӘselectionпјҲжҜ”еҰӮдёҖдёӘMoveпјҢдёҖдёӘPlanning EntityпјҢдёҖдёӘPlanning ValueпјүеңЁд»Җд№Ҳж—¶еҖҷиў«еҲӣе»әпјҢд»ҘеҸҠиҝҷдёӘselectionзҡ„з”ҹе‘Ҫе‘Ёжңҹ

еҹәжң¬дёҠжҜҸдёӘSelectorйғҪж”ҜжҢҒcacheTypeзҡ„и®ҫзҪ®

  • JUST_IN_TIMEпјҡй»ҳи®ӨеҖјпјҢеңЁеҚіе°Ҷз”ЁеҲ°жҹҗдёӘselectionд№ӢеүҚиҝӣиЎҢжһ„е»ә
  • STEPпјҡcachedпјҢеҲӣе»әдёҖдёӘselectionеңЁдёҖж¬Ўstepзҡ„ејҖеӨҙпјҢз„¶еҗҺжҠҠиҜҘselectionдҝқеӯҳдёӢжқҘпјҢдјҡжңүйўқеӨ–еҶ…еӯҳејҖй”Җ
  • PHASEпјҡcachedпјҢеҲӣе»әдёҖдёӘselectionеңЁдёҖдёӘsolver phaseзҡ„ејҖеӨҙпјҢз„¶еҗҺжҠҠиҜҘselectionдҝқеӯҳдёӢжқҘпјҢдјҡжңүйўқеӨ–еҶ…еӯҳејҖй”ҖпјҢдҪҶжҳҜжңүдёҖдәӣжҖ§иғҪдјҳеҢ–
  • SLVERпјҡcachedпјҢеҲӣе»әдёҖдёӘselectionеңЁsolverзҡ„ејҖеӨҙпјҢз„¶еҗҺжҠҠиҜҘselectionдҝқеӯҳдёӢжқҘпјҢдјҡжңүйўқеӨ–еҶ…еӯҳејҖй”ҖпјҢдҪҶжҳҜжңүдёҖдәӣжҖ§иғҪдјҳеҢ–
