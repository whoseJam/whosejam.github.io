Ubuntu中文输入法

2023-04-01 Linux Comments

输入框架

有ibus，fcitx等，需要下载准备好（ibus可能是ubuntu自带的，fcitx需要单独下载）

Input Sources

首先在Settings>Keyboard中找到Input Sources选项（根据版本不同，Input Sources也可能出现在Settings>Region & Language）

默认情况下可能这个列表中只有English (US)，将Chinese添加到这个列表中

IBus下配置中文输入法

Manage Installed Languages

在Settings>Region & Language中点击Manage Installed Language，同时注意到Keyboard input method system选项指向了IBus

把汉语(中国)挪动到顶上去

ibus-setup

在命令行中输入ibus-setup可以打开ibus的配置界面

Shortcut

在此处可以配置快捷键

Input Method

添加一种中文Input Method

（未走通）

fcitx下配置中文输入法

Manage Installed Languages

在Settings>Region & Language中点击Manage Installed Language，同时将Keyboard input method system选项指向了fcitx[4]

将汉语(中国)挪动到顶上去

最后Apply System-Wide

搜狗输入法

在搜狗官网上下载安装包，随后用dpkg安装

接着安装搜狗输入法的依赖

重启一下电脑

fcitx configure

在Input Method里面，保证存在搜狗拼音即可

修改系统默认语言

在Settings>Region & Language处修改

解决无法激活Language Support中的汉语

定义激活：将列表中某一项从下方移动到上方，使其文本从灰色变成黑色

如果把这个列表中按照分割线进行标号，English(United States)上面的那条线被标号为0，其下的一条线被标号为1，English下面的那条线被标号为2……

那么在拖动汉语(中国)的时候，将其拖动到1,2条线处都会导致激活失败，只有将其拖动到0处才能激活成功
